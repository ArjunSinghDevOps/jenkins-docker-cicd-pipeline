# Jenkins Docker CI/CD Pipeline Project

## Project Overview
This project demonstrates an end-to-end CI/CD pipeline using:

- GitHub
- Jenkins
- Maven
- Docker
- AWS EC2

## Workflow

GitHub Repository
        ↓
Jenkins Pipeline
        ↓
Maven Build
        ↓
Docker Image Build
        ↓
Docker Container Deployment
        ↓
AWS EC2

## Technologies Used

- Java Spring Boot
- Maven
- Jenkins
- Docker
- AWS EC2
- Git & GitHub
